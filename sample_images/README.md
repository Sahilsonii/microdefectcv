# Sample Images

Place your SEM or microstructure images here for testing.

## Recommended test images

| File | Description |
|---|---|
| `sem_pinhole_2d.png` | 2D perovskite SEM showing pinholes |
| `sem_pbi2_3d.png` | 3D perovskite SEM with PbI2 bright particles |
| `sem_mixed_3d_2d.png` | Mixed morphology image |

## Naming convention

Use descriptive names so the demo script output files are easy to identify.

## Notes

- Images should be **grayscale or BGR** (standard camera output).
- The filter automatically crops the SEM metadata bar at the bottom if present.
- Recommended image resolution: **512×512 to 2048×2048 px**.
